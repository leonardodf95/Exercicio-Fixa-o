Tags vistas até o momento:

Tag <h1> :
Tag para informar títulos, tambem existe as variações de h2 a h6, deixando os titulos cada vez menores.
Tag <p> :
Tag para formular parágrafos
Tag <a> :
Tag âncora, utilizada juntamente com o atributo 'href' para fazer Hiperligações com sites, arquivos, e-mails e telefones.
Tag <ol> :
Tag utilizada para criar listas ordenadas.
Tag <ul> :
Tag utilizada para criar listas não ordenadas.
Tag <li> :
Tag para os itens da lista criada.
Tag <br> :
Tag de quebra de linha.
Tag <div> :
Tag de criação de divisões no corpo do código HTML, tendo também uma pequena divisão na página em si.
Tag <img> :
Tag de inserção de imagens na página.
Tag <input> :
Tag de entrada de dados.
Tag <button> :
Tag para criação de botão.
Tag <strong> :
Tag que deixa o texto em negrito, mas sua utilização é para fins de acessibilidade em que os textos dentro dessa tag são lidos de forma mais intensa.
Tag <b> :
Tag que deixa o texto em negrito.
Tag <i> :
Tag que deixa o texto em itálico.
Tag <dl> :
Tag para criação de lista de descrição.
Tag <dt> :
Tag para os itens da lista de descrição.
Tag <dd> :
Tag para a descrição dos itens da 'lista de descrição'.
Tag <code> :
Tag para estilizar parte do texto especificando que ele é um código em programação.
Tag <blockquote> :
Tag para citações.
Tag <q> :
Tag para citações curtas.
